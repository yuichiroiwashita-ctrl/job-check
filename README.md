# 適職診断Webアプリ - 就活生向け

LINEリッチメニューから遷移する、就活生向けの適職診断Webアプリケーションです。

---

## 🎉 完成した機能

### ✅ 診断フロー（5ステップ）
1. **スタート画面** - 魅力的なヒーローデザイン
2. **MBTI選択** - 16タイプのカード選択
3. **質問（9問）** - 4択形式・プログレスバー付き
4. **ユーザー情報入力** - フォーム（LINE情報自動取得対応）
5. **結果表示** - ポップ・カラフルなスタイルB

### 🎨 スタイルB - ポップ・カラフルデザイン
- **タイプ別グラデーション背景** - 8タイプそれぞれに専用カラー
- **大きなタイプアイコン** - 絵文字で視覚的にわかりやすく
- **特性レーダーチャート** - Chart.jsによる6軸レーダーグラフ
- **強みバッジ** - タイプカラーに合わせた強み表示
- **アニメーション演出** - アイコン・バッジ・コンフェッティ

### 💚 LINE LIFF連携
- LINEアプリ内ブラウザで開いた場合、ユーザー情報を自動取得
- LINE UserID・表示名をフォームに自動入力
- LINEアプリ外からのアクセスでも正常動作

### 📊 Google スプレッドシート自動保存
- 診断完了時に自動保存（全21列）
- LINE情報・ユーザー入力・診断結果・全回答を記録

## 🛠 カスタマイズ方法

### 質問内容の変更
`index.html` の `questions` 配列を編集：
```javascript
const questions = [
  {
    text: '質問文',
    options: [
      { text: '選択肢1', type: 'leader' },
      { text: '選択肢2', type: 'creator' },
      // ...
    ]
  },
  // ...
];
```

### タイプ別カラーの変更
`typeConfig` オブジェクトを編集：
```javascript
strategist: {
  color1: '#6366F1',  // グラデーション開始色
  color2: '#8B5CF6',  // グラデーション終了色
  light: '#EEF2FF',   // 背景用の薄い色
  textColor: '#4338CA', // テキスト色
  // ...
}
```

### 企業情報の変更
`getResultData()` 関数内の `companies` 配列を編集：
```javascript
companies: [
  { 
    emoji: '💼', 
    name: '企業名', 
    industry: '業種', 
    comment: 'コメント' 
  },
  // ...
]
```

### Timerex URLの変更
各企業カードの `href` 属性を変更：
```javascript
card.href = 'https://timerex.net/s/あなたのURL';
```

## 🐛 トラブルシューティング

### 【重要】卒業予定年が反映されない場合のデバッグ手順

#### STEP 1: ブラウザでのデバッグ
1. ブラウザで`index.html`を開く
2. **F12**キーで開発者ツールを開く
3. **Console**タブを選択
4. 診断を進めて、ユーザー情報入力画面まで行く
5. フォームに情報を入力する
6. コンソールに以下を入力して Enter：
   ```javascript
   testFormValues()
   ```
7. 卒業予定年の値が表示されるか確認

#### STEP 2: 送信されるデータの確認
1. 「診断結果を見る」ボタンをクリック
2. コンソールに以下が表示されるか確認：
   ```
   submitUserInfo() が呼ばれました
   取得した値: {name: "...", graduation: "2025年3月", ...}
   送信するユーザーデータ: {...}
   スプレッドシートに送信するデータ: {...}
   ```

#### STEP 3: Google Apps Scriptのログ確認
1. [Google Apps Script](https://script.google.com/)を開く
2. 該当のスクリプトプロジェクトを開く
3. 左メニューの「実行数」をクリック
4. 最新の実行ログを確認
5. 「卒業予定年: 2025年3月」などと表示されているか確認

#### よくある原因と解決方法

| 問題 | 原因 | 解決方法 |
|---|---|---|
| コンソールに何も表示されない | JavaScriptエラー | F12 → Consoleタブでエラーを確認 |
| `graduation: ""` と空文字 | セレクトボックスで選択していない | 必ず選択肢を選ぶ |
| `graduation: undefined` | HTML要素IDが間違っている | `input-graduation`が正しいか確認 |
| スプシに列が追加されない | GASが古いバージョン | GASコードを最新版に更新してデプロイし直す |
| スプシ全体にデータが入らない | GAS Web App URLが間違っている | `index.html`のGAS_URLを確認 |

### スプレッドシートに保存されない
- GAS Web App URLが正しく設定されているか確認
- GASのデプロイ設定で「全員（匿名を含む）」を選択しているか確認
- ブラウザのコンソールでエラーを確認

### LINE情報が取得できない
- LIFF IDが正しく設定されているか確認
- LIFFのエンドポイントURLが公開URLと一致しているか確認
- LINEアプリ内ブラウザで開いているか確認

### レーダーチャートが表示されない
- Chart.jsのCDNが読み込まれているか確認（`<head>`内）
- ブラウザのコンソールでエラーを確認
